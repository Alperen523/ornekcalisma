# Giriş Bu dokümanda Vakko için sunulan API'nin açıklamaları, güvenlik kısıtları ve kullanım örnekleri sunulmaktadır.

API (Application Programming Interface), farklı yazılım uygulamalarının birbirleriyle etkileşime geçmelerine olanak sağlayan servis çözümüdür.
API'nin sunumunda amaç; Entegratörler için Vakko hizmetlerine erişim ve tümleşim ortamı sağlamaktır.

Tüm metotların HTTP istek ve cevap mesajlarında JSON söz dizim standardı kullanılmıştır.
Güvenli API'lerin erişiminde HTTP Authorization protokolü takip edilmektedir. Bu doğrultuda **Login > Authenticate** metodu ile alınan Token ile diğer metotlarla güvenli veri alışverişi sağlanmaktadır.
**Test Ortamı**
Link= http://crmapi-test.vakko.com.tr

**Canlı Ortam**
Link= http://crmapi.vakko.com.tr

Authorization işlemi için yetkililerden Username ve Password bilgisi alınmalıdır.
